### ➡ git 사용법 

``` bash
git stash	# 내가 하던 작업 임시저장(다른사람과 겹치지 않게)
git pull	# git 상황 보기(다른 사람 현황)
git stash pop	# 임시저장 풀기
git add "파일명"	# 작업한 파일 git에 추가
git commit -m "메세지"	# 커밋메시지 
git push	# git에 업로드 
```
